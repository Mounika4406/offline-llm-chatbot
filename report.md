## Results and Analysis

After running the chatbot with 20 customer queries, responses were generated using both zero-shot and one-shot prompting techniques. Each response was evaluated manually based on three criteria: relevance, coherence, and helpfulness.

The results showed that both prompting methods produced clear and understandable responses. However, the one-shot prompting technique performed slightly better overall. This is because the one-shot prompt includes an example response, which helps guide the model to produce answers in a more structured and helpful format.

Zero-shot responses were generally relevant but sometimes asked the user for more information rather than directly providing a solution. In contrast, the one-shot responses tended to provide clearer instructions and more complete answers.

Overall, the experiment demonstrates that providing an example within the prompt can improve the quality of responses generated by the model. This highlights the importance of prompt engineering when working with large language models.
